=========

Generates and sends notices to active Paid Memberships Pro members about their upcoming recurring payment.


== Description ==

 A more reliable and up-to-date notification system than the standard Recurring Payment Email Reminder add-on for Paid Memberships Pro. The E20R Payment Warning for PMPro plugin will utilize the most recent information generated by the payment gateway.

== Installation ==

1. Upload the `e20r-payment-warning-pmpro` directory to the `/wp-content/plugins/` directory of your site.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. That's it. Settings are managed through the "Settings" -> "E20R Roles for PMPro" options/settings page.
1. Licenses for the gateway modules(s) can be purchased on the Eighty/20 Results website (https://eighty20results.com/licenses)
